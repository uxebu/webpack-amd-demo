webpack-amd-demo
================

A simple webpack example with amd


To run this project locally do the following:
- `git clone git@github.com:uxebu/webpack-amd-demo.git` clone the repo on your machine
- `cd webpack-amd-demo` go into the directory where the project was cloned into
- `npm install` installs all dependencies into the `node_modules` directory
- `webpack` to execute `webpack.config.js` this creates the bundle.js file
